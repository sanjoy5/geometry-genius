What is DOM? What is the purpose of DOM?

Answer : DOM means Document Object Model. By using DOM we can create dynamic and interactive elements inside our HTML document. With the object model, javascript get all the power it needs to create dynamic HTML.

If we want to create any dynamic , interactive button event inside html, We can create it by DOM. We can use so many events like click, dblclick, mouseover, mouseout, mouseenter, keyup, keydown ect to make our website more dynamic or interactive by using DOM. Not only with button, we can make it with any type of HTML elements.

We can dynamicaly style our website using DOM. We can make css style via DOM. We can get and set and also change any place value using DOM. Using DOM we can get all control of any HTML elemnts by it's id, classes, tag names. We can change any eleemts attribute, classes ctc and also can set their value. DOM is best if you want to make a Dynamic website.

How can we select HTML elements using DOM?

Answer : Using DOM We can select HTML element in 5 different steps.

Step 1 : getElementById : Using 'getElementById' we can get HTML element id and can control it.
Example : document.getElementById('idName')

Step 2 : getElementsByClassName : Using 'getElementsByClassName' we can get HTML elements classes and can control them.
Example : document.getElementsByClassName('className')

Step 3 : getElementsByTagName : Using 'getElementsByTagName' we can get HTML elements tags and can control them.
Example : document.getElementsByTagName('tagName')

Step 4 : querySelector : Using 'querySelector' we can get HTML element id, class, tag and can control it.
Example : for class : document.querySelector('.className') | for id : document.querySelector('#idName') | for tag : document.querySelector('tagName')

Step 5 : querySelectorAll : Using 'querySelectorAll' we can get HTML elements ids, class, tags and can control them.
Example : for classes : document.querySelectorAll('.className') | for ids : document.querySelectorAll('#idName') | for tags : document.querySelectorAll('tagName')

What is Event Bubble?

Answer : Event Bubble is a DOM Event. Event Bubble is like a bubble in the water. If you register any event in a HTML element, this event will go through up to its parent element. This event will work for child element to its parent element.


For Example : Suppose, inside ul tag you have some li elements. if you register any event in any li elements. This event will go up li to ul. It means this event also trigger for ul element.
